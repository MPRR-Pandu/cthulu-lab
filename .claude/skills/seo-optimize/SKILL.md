# SEO Optimization

## When to Use
- Launching a new website
- Improving search rankings
- Adding meta tags and structured data
- Content optimization

## Instructions

1. Audit current page structure (headings, meta, content)
2. Identify target keywords
3. Optimize on-page elements
4. Add structured data (JSON-LD)
5. Check technical SEO (speed, mobile, crawlability)

## On-Page Checklist
- [ ] Title tag: under 60 chars, keyword first
- [ ] Meta description: under 155 chars, includes CTA
- [ ] H1: one per page, includes primary keyword
- [ ] H2-H3: logical hierarchy, include secondary keywords
- [ ] Image alt text: descriptive, includes keyword where natural
- [ ] Internal links: 3-5 per page
- [ ] URL: short, readable, includes keyword

## Technical Checklist
- [ ] Page load under 3 seconds
- [ ] Mobile responsive
- [ ] HTTPS
- [ ] Sitemap.xml exists
- [ ] robots.txt configured
- [ ] Canonical URLs set
- [ ] No broken links (404s)
- [ ] Structured data validates (schema.org)

## Rules
- Write for humans first, search engines second
- One primary keyword per page
- No keyword stuffing
- Include structured data for products, articles, FAQs
