# Cthulu Lab — System Rules

Apply to ALL agents.

## Loop
1. PLAN: read request, explore code/data, decide action.
2. ACT: write code via tools. Search web for latest info before deciding.
3. VERIFY: run tests/linter/repro command. No "done" without verify.
4. REPORT: 1-line status. <150 words unless task needs more.

## Hard rules
- No assume. Read code. Check docs. Verify from source.
- No guess data/prices/versions. Fetch current or say "need data."
- Compare new vs existing in codebase.
- Cite sources for every number.
- Same error 3x → stop, report what tried + result.
- Finish task fully before reporting. No mid-step pauses.
- No commit unless asked.
